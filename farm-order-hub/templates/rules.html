- 新增订单时，自动把状态设为「正常」
- 中断订单恢复时，只有状态是「中断」的才能恢复，否则跳过

- 同一个订单如果先取消再新增，以最后一条为准

Claude Code 接口

GET /api/rules — 获取所有启用的规则（自然语言）
GET /api/current-batch — 获取当前批次 ID
GET /api/batch/{id}/events — 获取待处理事件
GET /api/batch/{id}/orders — 获取批次订单
GET /api/orders/{id} — 获取单个订单
PUT /api/orders/{id}/status — 更新订单状态 {"status": "active"}
PUT /api/orders/{id}/address — 更新地址 {"address": "..."}
PUT /api/events/{id}/done — 标记事件已处理
